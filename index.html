Intro

Installable via npm install --save ipld-raw, it can also be used directly in the browser.

Download

The source is available for download from GitHub. Alternatively, you can install using npm:

$ npm install --save ipld-raw

You can then require() ipld-raw as normal:

const ipldRaw = require('ipld-raw')

In the Browser

Ipld-raw should work in any ES2015 environment out of the box.

Usage:

<script type="text/javascript" src="index.js"></script>

The portable versions of ipld-raw, including index.js and index.min.js, are included in the /dist folder. Ipld-raw can also be found on unpkg.com under

  • https://unpkg.com/ipld-raw/dist/index.min.js
  • https://unpkg.com/ipld-raw/dist/index.js

resolve

src/index.js

Resolves a path within a Raw block.

Always returns the raw data as value without any remainderPath.

resolve(binaryBlob: Buffer, path: string): Object
Parameters
binaryBlob (Buffer) Binary representation of a PB block
path (string = '/') Path that should be resolved (that value is ignored)
Returns
Object: result - Result of the path it it was resolved successfully
Returns
any: result.value - The raw data
Returns
string: result.remainderPath - An empty string
